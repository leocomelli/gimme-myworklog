# Gimme my worklog

This project will create a csv file with all worklog registered at Jira (Tempo plugin).

### Usage:

	main -url=<JIRA_URL> \
	     -startDate=<YYYY-MM-DD> \
	     -endDate=<YYYY-MM-DD> \
	     -tempoApiToken=<TEMPO_API_TOKEN> \
	     -username=<USERNAME> \
	     -password=<PASSWORD> \
	     -output=<OUTPUT_FILE>
